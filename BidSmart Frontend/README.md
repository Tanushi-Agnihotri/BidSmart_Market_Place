# BidSmart Frontend

The frontend for BidSmart — an intelligent online auction platform. Bid smarter, win faster.

## Getting started

Requires Node.js & npm installed ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

```sh
# 1. Clone the repository
git clone <YOUR_GIT_URL>

# 2. Enter the project directory
cd "BidSmart Frontend"

# 3. Install dependencies
npm i

# 4. Start the dev server
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
